## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/theharsh999/harsh-patel-portfolio.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Create Production Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
