## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/theharsh999/harsh-patel-portfolio.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
